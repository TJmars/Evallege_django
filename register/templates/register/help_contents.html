Evallegeの使い方

①授業の評価をみる

Evallegeでは、「学年比率」「成績」「難易度」「課題量」の情報を提供しています。これらの情報を参考に、自分にあった授業を探してみましょう！

１.　授業の評価をみる

Evallegeでは、「学年比率」「成績」「難易度」「課題量」の情報を提供しています。これらの情報を参考に、自分にあった授業を探してみましょう！

２．授業内チャットを活用する

授業内チャットを使って、同じ授業を履修している仲間にさまざまな質問をすることができます。難しい課題が出た際などにはぜひ活用してみましょう！ このチャットは履修前にも閲覧できるため、履修時にもご活用いただけます。

３．授業の評価をする

成績が発表されたら、授業評価をしましょう！　評価を投稿することでさまざまな特典を受けることができます。個人の成績等回答内容が公開されることはないので ご安心ください。

４．教科書を売買する

履修が終わった授業の教科書を販売することができます。大学内での販売のため、学校で受け渡しを行えば、送料などは一切かかりません。
※近日実装予定です。

{% elif help_pk == 1 %}

履修講義の登録/解除について

  • ① 画面上部の「講義を探す」タブをプッシュ
  • ② 授業名、または教員名を入力して検索
  • ③ 登録ボタンをプッシュ
  • ④ 画面上部の「履修講義一覧」をプッシュすると、履修中の講義が見れます

{% elif help_pk == 2 %}

履修している講義がない場合

  • ① 画面上部の「新しい講義を作成」タブをプッシュ
  • ② 授業名、教員名を入力して講義を作成
  • ＊ 授業名、教員名がともに一致する講義がある場合、作成することができません。 講義を作成する前に検索ページで該当する講義がないか確認してください。
  • ④ ＊ 授業は登録されている講義名を参考に、正式名を入力してください。 教員名は、姓名の間にスペースを入れてください。

{% elif help_pk == 3 %}

チャット機能について

  • ① 講義リストの「チャット」ボタンをプッシュ
  • ② チャット画面で授業に関するさまざまな質問ができます。ぜひお気軽に質問してください。
  • ＊ 投稿内容には十分お気をつけください。不適切な投稿であると判断された場合、 ユーザーの許可を得ず、投稿の削除、ユーザーの抹消等の措置を行います。

{% elif help_pk == 4 %}

詳細画面・グラフの見方について

  • ① 学年比率グラフ

    　　当該講義を履修した生徒の学年を表示します。
  • ② 成績グラフ

    　　当該講義を履修した生徒の成績比率を表示します。
  • ③ 難易度グラフ

    　　当該授業の履修生の難易度に関する回答結果を表示します。
  • ④ 課題量グラフ

    　　当該授業の履修生の難易度に関する回答結果を表示します。
  • ④ 授業方式切り替え

    　　対面、またはオンラインを選択することで、授業方式を限定したグラフを表示することができます。

{% elif help_pk == 5 %}

授業評価について

  • 学年

    　　当該講義を履修した学年を選択してください。
  • 成績

    　　当該講義の成績を選択してください。(該当する成績がない場合、Dとしてください。)
  • 課題量

    　　課題の量について体感に最も近いものを選択してください。
  • 授業の難易度

    　　当該授業の難易度について体感に最も近いものを選択してください。
  • 授業方式

    　　当該授業を履修した方式を選択してください。
  • コメント

    　　その他、履修に関する有益な情報等ございましたら、入力してください。
  • 　　＊　同一の講義に対する評価は一度しか行えません。また、履修登録後1ヶ月が経過した講義は、 評価を投稿するまで解除することができません。

ユーザー情報閲覧について

ユーザーネーム変更について

新しいユーザーネームを入力し、送信ボタンをプッシュしてください。 ユーザーネームはチャット画面での投稿者名として表示されます。

メールアドレス変更について

新しいメールアドレスを入力し、送信ボタンをプッシュしてください。 新しいメールアドレスに送信されたURLをタップすることで変更が適用されます。

パスワード変更について

現在のパスワードと新しいパスワードを入力し、送信ボタンをプッシュしてください。
